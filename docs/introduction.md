# Introduction

Radix is a lightweight, local-first command-line search engine designed for developers who value simplicity, performance, and control. Unlike traditional search solutions like Elasticsearch, Radix operates entirely locally, leveraging SQLite's FTS5 extension for high-performance full-text search.

## Why Radix?

- **Local-First**: No external servers or dependencies.
- **Lightweight**: Minimal resource usage.
- **Portable**: Works anywhere SQLite works.
- **Customizable**: Supports `.radixignore` for fine-grained control over indexing.

## Core Philosophy

Radix was built to provide a fast, reliable, and local alternative to heavyweight search solutions. By focusing on SQLite and Python, Radix ensures minimal dependencies and maximum portability.
