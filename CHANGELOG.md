# Changelog

## [0.1.0] - 2026-03-18
### Added
- Initial release with the following features:
  - `radix init`: Initialize the database.
  - `radix index`: Index files in a directory.
  - `radix search`: Search the index.
  - `radix status`: Show database status.
  - `radix doctor`: Check database health.
  - `--watch` mode for `index`.
  - `--json` output for `search`.
